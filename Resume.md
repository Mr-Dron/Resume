### Authorization System (Backend API)

**Стек**: FastAPI · SQLAlchemy (async) · Alembic · JWT · Docker · SQLite(с возможностью замены на PostgreSQL)

Асинхронный backend-сервис с собственной реализацией системы авторизации и аутентификации:
- Регистрация и аутентификация пользователей
- Logout и принудительное разлогинивание неактивных пользователей
- JWT Access + Refresh токены
- Soft delete пользователей и сущностей с хранением данных заданный период
- Ролевая модель доступа (RBAC)
- Проверка прав доступа через FastAPI dependencies
- Отслеживание активности пользователя (online/offline)

🔗 **GitHub:** https://github.com/Mr-Dron/authorization-system
